## Known Patterns / Lessons Learned
- Wave requires paid plan to link bank account — do not use for new Ellie sites
- Global string replacement in PowerShell: use `[System.IO.File]::ReadAllText` not `Get-Content -Raw`
- Robocopy exit code 1 = success with minor warnings, not a failure
- The style-guide, capsule-wardrobe, how-to-dress, fashion-subscription pages are high-traffic SEO pages — don't delete them
- New Ellie sites must be built as separate legal entities (separate LLC, EIN, bank, Stripe)
